## よくある質問

#### docker-compose.ymlに書いてあるsecretsがないんだけど？
secretsディレクトリとその中身は自分で作成しよう！
パスワードはインターネットに残せないからローカルのファイルを参照する仕組みになっているよ

#### docker-compose.ymlにある.envがないんだけど？
コレも自分で作ろう。ホストに環境変数が設定されてなかった場合、dockerがここから取得してくれるらしい

#### volumeが消えた！
今の書き方は消えない設定になっているはず。もう一度マウント先を確認しよう。
それでも見つからない場合、なにか書き換えて消えるボリューム設定になっているかも。
